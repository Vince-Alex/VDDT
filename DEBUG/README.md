# VDDT - 多功能视频下载与转码工具 (v2.1.0)

**VDDT (Video Downloader & Transcoder Tool)** 是一个基于 `yt-dlp` 和 `FFmpeg` 构建的高效、可视化视频处理工具。它专为追求终端操作效率的用户设计，特别针对 **Termux (Android)** 和标准 Linux 环境进行了深度优化。

> **核心亮点**: 全新的 TUI 图形界面，支持可视化文件浏览，告别繁琐的路径输入。

---

## 🚀 核心特性

- **🌍 全能下载**: 支持全球 1000+ 视频网站（Bilibili, YouTube, TikTok, Douyin 等）。
- **🖥️ TUI 图形界面**: 完整的终端菜单导航系统，无需记忆复杂命令。
- **📁 可视化文件浏览器**: 离线转码功能内置文件浏览器，支持方向键穿梭目录，一键选择文件或文件夹。
- **📊 实时进度监控**: 引入 `ffmpeg-progress-yield`，精准显示单个文件内部的转码百分比，进度丝滑不卡顿。
- **⚙️ 离线转码矩阵**:
  - **无损封装**: `Copy/Remux` 模式，秒级完成格式转换。
  - **高清预设**: 自动压缩至 720p/1080p (H.264)。
  - **音频提取**: 一键提取 MP3 (192kbps) 高质音频。
  - **复古兼容**: 专为老式播放器提供的 `AMV` 格式支持。
- **📝 智能命名**: 下载及转码文件自动附加 `[VDDT]` 标识，日期/作者/标题信息完整保留。

---

## 🛠️ 安装与依赖

### 1. 环境要求
- **Python**: 3.8+
- **FFmpeg**: 系统需安装 FFmpeg 且已加入环境变量。

### 2. 快速安装
```bash
# 克隆仓库
git clone https://github.com/Vince-Alex/VDDT.git
cd VDDT

# 安装必要依赖
pip install yt-dlp colorama ffmpeg-progress-yield tqdm requests
```

## 📖 使用指南

### 启动 TUI 模式 (推荐)
直接进入功能最全、交互最友好的调试版界面：
```bash
python3 DEBUG/main.py
```

### 快捷键说明
| 按键 | 功能描述 |
| :--- | :--- |
| **↑ / ↓ (j/k)** | 菜单导航 / 文件列表翻页 |
| **Enter (回车)** | 确认选择 / 进入文件夹 |
| **Q / q** | **返回 / 取消 / 退出** (替代原 ESC 键) |
| **S / s** | 在文件浏览器中**选定当前目录** |
| **1 - 6** | 菜单项快速数字键选择 |
| **H / h** | 显示详细帮助弹窗 |
| **Ctrl + C** | 强制终止程序 |

## 📂 项目结构

- `DEBUG/`: **核心开发版**。包含最新的 TUI 界面、可视化浏览器及实时进度逻辑。
- `VDDT-beta/`: 实验性功能分支。
- `cookies/`: 放置 `.ck` 文件以支持会员视频或受限视频下载。
- `VDDT_Downloads/`: 默认下载与转码输出目录。

---

## ⚠️ 注意事项

- **Termux 用户**: 若界面渲染异常，请尝试执行 `export TERM=xterm-256color` 后再次启动。
- **Cookie 说明**: 将浏览器导出的 Cookie 存放在 `cookies/` 目录下，命名为 `域名.ck`（如 `bilibili.com.ck`），程序将自动加载。
